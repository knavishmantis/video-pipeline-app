# KnavishMantis Script Writing Guide

This directory contains all reference material for writing scripts and scene breakdowns for the KnavishMantis Minecraft YouTube Shorts channel.

## Structure

- `frameworks/` — Script writing frameworks and techniques
  - `knavishmantis-voice.md` — Channel identity, script formula, scene breakdown guide
  - `kallaway-frameworks.md` — Kallaway's short-form video frameworks (hooks, story locks, curiosity loops)
- `competitor-data/` — Downloaded transcripts from competitor channels
- `analysis/` — AI-generated analysis of competitor patterns

## How to use

When Claude is asked to help write or improve a script, it should reference the files in `frameworks/` for guidance on voice, structure, and technique.
